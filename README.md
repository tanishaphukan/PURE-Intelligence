# 🧪 PURE Intelligence - Heavy Metal Pollution Monitor

> Automated web platform for calculating water pollution indices, mapping contamination zones, and predicting health risks using ML.

**Smart India Hackathon 2025** | Problem ID: SIH25067 | Team: The Alchemists

---

## What It Does

Eliminates manual computation of heavy metal pollution indices by providing a single centralized platform - with interactive maps, ML-based disease prediction, and a gamified leaderboard for community engagement.

---

## Key Features

- 📊 Auto-calculates **HPI, HEI & Contamination Degree**
- 🗺️ **Interactive pollution heatmap** via Leaflet.js
- 🤖 **ML health risk prediction** for disease likelihood
- 🏆 **Gamified leaderboard** ranking locations by risk
- 💬 **AI chatbot** for user guidance
- 📤 **CSV upload/export** for bulk data
- 🚦 Color-coded risk indicators for non-technical users

## Pollution Indices

| Index | Safe Limit |
|---|---|
| HPI (Heavy Metal Pollution Index) | < 15 |
| HEI (Heavy Metal Evaluation Index) | < 10 |
| Cd (Contamination Degree) | < 5 |
